List of Flags¶

PyDesigner is extremely flexible when it comes to dMRI processing. Users can easily enable or disable various preocessing steps without changing the overall sequence.

THe list below covers all these flags.

Flag

Usage

-o, --output

pydesigner output directory

-s, --standard

runs the recommended preprocessing pipeline in order: denoise, degibbs, undistort, brain mask, smooth, rician

Preprocessing Control

--denoise

denoises input DWI

--extent

shape of denoising extent matrix; default: 5,5,5

--degibbs

corrects Gibb’s ringing

--undistort

undistorts image using a suite of EPI distortion correction,

eddy current correction, and co-registration. Does not run EPI correction if reverse phase encoding DWI is absent

--epi

speeds up topup if a reverse PE is present; specify the number (integer) of reverse PE direction B0 pairs to use

--mask

computes a brain mask at 0.20 threshold

--maskthr

specify FSL bet fractional intensity threshold for brain masking; default: 0.20

--user_mask

provide path to user-generated brain mask in NifTi (.nii) format

--smooth

smooths DWI data at a default FWHM of 1.25

--fwhm

specify the FWHM at which to smooth; default 1.25

--rician

corrects Rician bias

Postprocessing Control

--nofit

performs preprocessing only; disables DTI/DKI parameter extraction

--noakc

disables brute forced kurtosis tensor outlier rejection

--nooutliers

disables IRLLS outlier detection

-w, --wmti

computes white matter tract integrity (WMTI) metrics

--fit_constraints

specify fitting constraints to use; default: 0,1,0

--noqc

disables saving of quality control (QC) metrics

--median

performs post processing median filter of final DTI/DKI maps

System Control

--nthreads

specify number of CPU workers to use in processing; default: all

--resume

resumes preprocessing from an aborted or partial previous run

--force

forces overwrite of existing output files

--verbose

displays console output

--adv

disables safety check to force run certain preprocessing steps THIS FLAG IS FOR ADVANCED USERS ONLY WHO FULLY ‘,’UNDERSTAND THE MRI SYSTEM AND ITS OUTPUTS. ‘,’RUNNING WITH THIS FLAG COULD POTENTIALLY ‘,’RESULT IN IMPRECISE AND INACCURATE RESULTS.’
